## Documentation officielle

### Pages de manuel

Les pages `man` restent la référence la plus fiable pour toute commande Linux. Depuis un terminal :

```bash
man ls
man 5 fstab
man -k network
```

En ligne, plusieurs sites proposent les man pages dans un format lisible :

- [man7.org](https://man7.org/linux/man-pages/) — pages de manuel Linux officielles
- [manpages.ubuntu.com](https://manpages.ubuntu.com/) — pages de manuel Ubuntu
- [die.net](https://linux.die.net/man/) — interface de recherche rapide

### Standards et specifications

- [The Open Group — POSIX](https://pubs.opengroup.org/onlinepubs/9699919799/) — specification complète du standard POSIX
- [Filesystem Hierarchy Standard](https://refspecs.linuxfoundation.org/FHS_3.0/fhs/index.html) — organisation standard des répertoires Linux
- [Linux Standard Base](https://refspecs.linuxfoundation.org/lsb.shtml) — specification de compatibilité entre distributions

## Outils indispensables

### Editeurs de texte

| Editeur     | Type      | Utilisation                                    |
| ----------- | --------- | ---------------------------------------------- |
| **Vim**     | Terminal  | Edition rapide, scripts, configuration serveur |
| **Nano**    | Terminal  | Edition simple, ideal pour les débutants       |
| **VS Code** | Graphique | Developpement, extensions SSH Remote           |
| **Emacs**   | Les deux  | Edition avancee, extensible en Lisp            |

### Outils de diagnostic

| Outil     | Usage                                         |
| --------- | --------------------------------------------- |
| `htop`    | Surveillancé interactive des processus        |
| `iotop`   | Surveillancé des entrées/sorties disque       |
| `iftop`   | Surveillancé du trafic réseau en temps reel   |
| `ncdu`    | Analyse de l'utilisation disque               |
| `strace`  | Tracer les appels système d'un processus      |
| `tcpdump` | Capture et analyse de paquets réseau          |
| `lsof`    | Lister les fichiers ouverts par les processus |

### Gestionnaires de paquets

| Distribution  | Gestionnaire | Commande d'installation      |
| ------------- | ------------ | ---------------------------- |
| Debian/Ubuntu | APT          | `sudo apt install paquet`    |
| Fedora/RHEL   | DNF          | `sudo dnf install paquet`    |
| Arch Linux    | Pacman       | `sudo pacman -S paquet`      |
| openSUSE      | Zypper       | `sudo zypper install paquet` |
| Alpine        | APK          | `sudo apk add paquet`        |

## Distributions Linux

### Pour debuter

- [Ubuntu](https://ubuntu.com/) — la plus populaire, large communaute
- [Linux Mint](https://linuxmint.com/) — basée sur Ubuntu, interface familiere
- [Fedora](https://fedoraproject.org/) — technologies recentes, soutenue par Red Hat
- [Pop!\_OS](https://pop.system76.com/) — optimisee pour le développement et le gaming

### Pour serveurs

- [Debian](https://www.debian.org/) — stabilité et fiabilité, base de nombreuses distributions
- [Ubuntu Server](https://ubuntu.com/server) — large écosystème, bonne documentation
- [Rocky Linux](https://rockylinux.org/) — compatible RHEL, remplacant de CentOS
- [AlmaLinux](https://almalinux.org/) — alternative a CentOS, compatible RHEL

### Pour utilisateurs avances

- [Arch Linux](https://archlinux.org/) — rolling release, installation manuelle, personnalisation totale
- [Gentoo](https://www.gentoo.org/) — compilation depuis les sources, contrôle maximal
- [NixOS](https://nixos.org/) — configuration declarative, reproductible
- [Void Linux](https://voidlinux.org/) — indépendante, init runit, minimaliste

## RFC et standards réseau

Les RFC (Request For Comments) definissent les protocoles Internet :

| RFC      | Protocole | Description                   |
| -------- | --------- | ----------------------------- |
| RFC 791  | IP        | Internet Protocol             |
| RFC 793  | TCP       | Transmission Control Protocol |
| RFC 768  | UDP       | User Datagram Protocol        |
| RFC 2616 | HTTP/1.1  | Hypertext Transfer Protocol   |
| RFC 4251 | SSH       | Secure Shell Protocol         |
| RFC 5321 | SMTP      | Simple Mail Transfer Protocol |
| RFC 1035 | DNS       | Domain Name System            |

Consulter les RFC : [rfc-editor.org](https://www.rfc-editor.org/)
