## Introduction

Sous Linux, les logiciels sont distribués sous forme de **paquets**. Chaque famille de distribution dispose de son propre gestionnaire de paquets. Ce tutoriel couvre les quatre principaux gestionnaires ainsi que les formats universels Snap et Flatpak.

## APT -- Debian, Ubuntu et derives

APT (Advanced Package Tool) est le gestionnaire de paquets des distributions basées sur Debian.

### Mettre a jour la liste des paquets et le système

```bash
sudo apt update
sudo apt upgrade -y
```

### Rechercher un paquet

```bash
apt search nginx
apt show nginx
```

### Installer et supprimer un paquet

```bash
sudo apt install nginx -y
sudo apt remove nginx
sudo apt purge nginx        # supprimé aussi les fichiers de configuration
sudo apt autoremove          # supprimé les dépendances orphelines
```

### Lister les paquets installés

```bash
dpkg -l
apt list --installed
```

### Les dépôts (repositories)

Les sources de paquets sont définies dans `/etc/apt/sources.list` et dans le répertoire `/etc/apt/sources.list.d/`. Pour ajouter un dépôt externe :

```bash
sudo add-apt-repository ppa:ondrej/php
sudo apt update
```

## DNF -- Fedora, RHEL 9+ et derives

DNF a remplacé YUM comme gestionnaire par défaut sur les distributions Red Hat.

### Operations courantes

```bash
sudo dnf check-update
sudo dnf upgrade -y
dnf search httpd
sudo dnf install httpd -y
sudo dnf remove httpd
dnf list installed
```

### Gestion des dépôts

```bash
dnf repolist
sudo dnf config-manager --add-repo https://example.com/repo.repo
```

## Pacman -- Arch Linux et derives

Pacman utilisé une syntaxe compacte a base de drapeaux.

| Action | Commande |
|---|---|
| Synchroniser et mettre a jour | `sudo pacman -Syu` |
| Rechercher un paquet | `pacman -Ss vim` |
| Installer | `sudo pacman -S vim` |
| Supprimer avec dépendances | `sudo pacman -Rns vim` |
| Lister les paquets installés | `pacman -Q` |

### Exemple complet

```bash
sudo pacman -Syu
sudo pacman -S git base-devel
```

## APK -- Alpine Linux

Alpine utilisé APK, un gestionnaire léger conçu pour les environnements minimaux et les conteneurs.

```bash
apk update
apk search curl
apk add curl
apk del curl
apk list --installed
```

## Comprendre les dépendances

Lorsque vous installez un paquet, le gestionnaire resout automatiquement ses **dépendances** -- les autres paquets nécessaires a son fonctionnement. Par exemple :

```bash
apt depends nginx
dnf deplist httpd
```

Si une dépendance est absente, le gestionnaire la téléchargé et l'installe. Lors de la suppression, les dépendances orphelines peuvent être nettoyees avec `apt autoremove` ou `dnf autoremove`.

## Snap et Flatpak -- Paquets universels

### Snap (Canonical)

```bash
sudo apt install snapd
sudo snap install vlc
snap list
sudo snap remove vlc
```

### Flatpak

```bash
sudo apt install flatpak
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak install flathub org.gimp.GIMP
flatpak list
flatpak uninstall org.gimp.GIMP
```

Les paquets Snap et Flatpak sont isoles du système (sandboxing) et embarquent leurs propres dépendances, ce qui garantit la compatibilité entre distributions mais augmente l'espace disque utilise.

## Resume des commandes

| Action | APT | DNF | Pacman | APK |
|---|---|---|---|---|
| Mettre a jour les sources | `apt update` | `dnf check-update` | `pacman -Sy` | `apk update` |
| Mettre a jour le système | `apt upgrade` | `dnf upgrade` | `pacman -Syu` | `apk upgrade` |
| Installer | `apt install` | `dnf install` | `pacman -S` | `apk add` |
| Supprimer | `apt remove` | `dnf remove` | `pacman -R` | `apk del` |
| Rechercher | `apt search` | `dnf search` | `pacman -Ss` | `apk search` |

## Conclusion

La maitrise du gestionnaire de paquets de votre distribution est une competence fondamentale. Quel que soit le système que vous utilisez, les concepts restent les memes : rechercher, installer, mettre a jour et supprimer proprement les logiciels.
