## Syntaxe

```bash
ss [OPTIONS] [FILTRE]
```

## Options principales

| Option | Description                                       |
| ------ | ------------------------------------------------- |
| `-t`   | Afficher les connexions TCP                       |
| `-u`   | Afficher les connexions UDP                       |
| `-l`   | Afficher uniquement les sockets en ecoute         |
| `-n`   | Afficher les numeros de port sans résolution       |
| `-p`   | Afficher le processus associé a chaque socket     |
| `-a`   | Afficher toutes les connexions                    |
| `-s`   | Afficher les statistiques resumees                |

## Exemples

### Lister les ports TCP en ecoute

```bash
$ ss -tlnp
State   Recv-Q  Send-Q   Local Address:Port   Peer Address:Port  Process
LISTEN  0       128      0.0.0.0:22            0.0.0.0:*          users:(("sshd",pid=842,fd=3))
LISTEN  0       511      0.0.0.0:80            0.0.0.0:*          users:(("nginx",pid=1205,fd=6))
LISTEN  0       128      0.0.0.0:443           0.0.0.0:*          users:(("nginx",pid=1205,fd=7))
```

### Afficher toutes les connexions TCP etablies

```bash
$ ss -tn state established
Recv-Q  Send-Q   Local Address:Port   Peer Address:Port
0       0        192.168.1.10:22      192.168.1.50:54312
0       0        192.168.1.10:80      10.0.0.5:48721
0       36       192.168.1.10:22      192.168.1.50:54298
```

### Afficher les statistiques resumees

```bash
$ ss -s
Total: 245
TCP:   12 (estab 3, closed 2, orphaned 0, timewait 1)

Transport  Total   IP     IPv6
RAW        1       0      1
UDP        5       3      2
TCP        10      7      3
INET       16      10     6
```

### Filtrer par port de destination

```bash
$ ss -tn dst :443
State   Recv-Q  Send-Q   Local Address:Port   Peer Address:Port
ESTAB   0       0        192.168.1.10:52341   93.184.216.34:443
ESTAB   0       0        192.168.1.10:52387   142.250.75.238:443
```
