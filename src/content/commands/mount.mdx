## Syntaxe

```bash
mount [OPTIONS] [PERIPHERIQUE] [POINT_DE_MONTAGE]
```

## Options principales

| Option       | Description                                      |
| ------------ | ------------------------------------------------ |
| `-t TYPE`    | Spécifier le type de système de fichiers          |
| `-o OPTIONS` | Options de montage (ro, rw, noexec, etc.)        |
| `-a`         | Monter tous les systèmes de fichiers de /etc/fstab |
| `-l`         | Lister les systèmes de fichiers montés avec labels |

## Exemples

### Lister tous les systèmes de fichiers montés

```bash
$ mount | column -t
/dev/sda1   on  /         type  ext4    (rw,relatime,errors=remount-ro)
tmpfs       on  /run      type  tmpfs   (rw,nosuid,nodev,noexec,relatime)
/dev/sdb1   on  /home     type  ext4    (rw,relatime)
/dev/sdc1   on  /mnt/data type  xfs     (rw,relatime)
```

### Monter une clé USB en lecture seule

```bash
$ sudo mount -t vfat -o ro /dev/sdb1 /mnt/usb
$ ls /mnt/usb/
documents/  photos/  backup.tar.gz
```

### Monter une image ISO

```bash
$ sudo mount -o loop /home/mouctar/debian-12.iso /mnt/iso
$ ls /mnt/iso/
boot/  dists/  install/  pool/  README.txt
```
