## Syntaxe

```bash
diff [OPTIONS] FICHIER1 FICHIER2
```

## Options principales

| Option | Description                                         |
| ------ | --------------------------------------------------- |
| `-u`   | Afficher en format unifié (contexte de 3 lignes)    |
| `-r`   | Comparer récursivement les répertoires              |
| `-i`   | Ignorer les différences de casse                    |
| `--color` | Afficher les différences avec des couleurs       |

## Exemples

### Comparer deux fichiers de configuration

```bash
$ diff config-prod.yml config-dev.yml
3c3
< port: 443
---
> port: 3000
7,8c7,8
< log_level: warn
< debug: false
---
> log_level: debug
> debug: true
```

### Afficher les différences en format unifié

```bash
$ diff -u ancien.txt nouveau.txt
--- ancien.txt	2026-05-07 14:30:00
+++ nouveau.txt	2026-05-08 09:15:00
@@ -1,5 +1,5 @@
 Serveur principal
-adresse: 192.168.1.10
+adresse: 10.0.0.50
 port: 8080
-timeout: 30
+timeout: 60
 actif: oui
```

### Comparer deux répertoires récursivement

```bash
$ diff -r src/ src-backup/
diff -r src/index.ts src-backup/index.ts
5c5
< const VERSION = "2.1.0";
---
> const VERSION = "2.0.0";
Only in src/: nouveau-module.ts
Only in src-backup/: ancien-module.ts
```
