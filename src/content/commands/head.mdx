## Syntaxe

```bash
head [OPTIONS] [FICHIER...]
```

## Options principales

| Option | Description                                    |
| ------ | ---------------------------------------------- |
| `-n`   | Afficher les N premières lignes                |
| `-c`   | Afficher les N premiers octets                 |
| `-q`   | Ne pas afficher les en-têtes de noms de fichier|
| `-v`   | Toujours afficher les en-têtes de noms de fichier |

## Exemples

### Afficher les 5 premières lignes d'un fichier

```bash
$ head -n 5 /etc/passwd
root:x:0:0:root:/root:/bin/bash
daemon:x:1:1:daemon:/usr/sbin:/usr/sbin/nologin
bin:x:2:2:bin:/bin:/usr/sbin/nologin
sys:x:3:3:sys:/dev:/usr/sbin/nologin
sync:x:4:65534:sync:/bin:/bin/sync
```

### Afficher les 100 premiers octets d'un fichier

```bash
$ head -c 100 /var/log/syslog
Mai  8 10:00:01 serveur-web-01 systemd[1]: Started Daily apt download activities.
```

### Comparer le début de plusieurs fichiers

```bash
$ head -n 3 access.log error.log
==> access.log <==
192.168.1.10 - - [08/Mai/2026:10:00:01] "GET / HTTP/1.1" 200 4523
192.168.1.15 - - [08/Mai/2026:10:00:03] "GET /api/users HTTP/1.1" 200 1287
192.168.1.22 - - [08/Mai/2026:10:00:05] "POST /login HTTP/1.1" 302 0

==> error.log <==
[Mai 08 10:00:02] [error] ModSecurity: Access denied with code 403
[Mai 08 10:01:15] [warn] Connection timeout from 10.0.0.5
[Mai 08 10:02:30] [error] File not found: /var/www/html/favicon.ico
```
