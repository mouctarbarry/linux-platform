## Syntaxe

```bash
env [OPTIONS] [NOM=VALEUR...] [COMMANDE]
```

## Options principales

| Option       | Description                                           |
| ------------ | ----------------------------------------------------- |
| `-i`         | Démarrer avec un environnement vide                   |
| `-u NOM`     | Supprimer la variable NOM de l'environnement          |
| `NOM=VALEUR` | Définir une variable avant d'exécuter la commande     |
| `-0`         | Séparer les entrées par un octet nul au lieu d'un saut de ligne |

## Exemples

### Afficher toutes les variables d'environnement

```bash
$ env
HOME=/home/mouctar
USER=mouctar
SHELL=/bin/bash
LANG=fr_FR.UTF-8
PATH=/usr/local/bin:/usr/bin:/bin
TERM=xterm-256color
```

### Exécuter une commande avec une variable modifiée

```bash
$ env LANG=en_US.UTF-8 date
Thu May  8 14:32:18 CEST 2025
```

### Exécuter une commande dans un environnement vide

```bash
$ env -i HOME=/tmp bash -c 'echo $HOME; echo $USER'
/tmp

```
