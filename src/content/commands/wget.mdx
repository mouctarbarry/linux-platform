## Syntaxe

```bash
wget [OPTIONS] URL
```

## Options principales

| Option | Description                                      |
| ------ | ------------------------------------------------ |
| `-O`   | Spécifier le nom du fichier de sortie            |
| `-q`   | Mode silencieux sans affichage de progrèssion    |
| `-c`   | Reprendre un téléchargement interrompu           |
| `-r`   | Télécharger récursivement (miroir de site)       |
