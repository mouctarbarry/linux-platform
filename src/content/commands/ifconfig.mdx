## Syntaxe

```bash
ifconfig [INTERFACE] [OPTIONS]
```

## Options principales

| Option      | Description                                    |
| ----------- | ---------------------------------------------- |
| `up`        | Activer une interface réseau                   |
| `down`      | Désactiver une interface réseau                |
| `netmask`   | Définir le masque de sous-réseau               |
| `-a`        | Afficher toutes les interfaces, même inactives |
