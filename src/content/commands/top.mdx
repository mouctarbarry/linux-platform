## Syntaxe

```bash
top [OPTIONS]
```

## Options principales

| Option | Description                                      |
| ------ | ------------------------------------------------ |
| `-d`   | Définir le délai de rafraîchissement en secondes |
| `-u`   | Filtrer par utilisateur                          |
| `-p`   | Surveiller uniquement les PID spécifiés          |
| `-n`   | Nombre d'itérations avant de quitter             |

```bash
$ top -u mouctar -n 1

  PID USER      PR  NI    VIRT    RES    SHR S  %CPU  %MEM     TIME+ COMMAND

 2103 mouctar   20   0 1876432  98764  32456 S   1.2   0.6   0:46.12 code
 2450 mouctar   20   0  654321  34567  12345 S   0.3   0.2   0:12.34 bash
```

```bash
$ top -p 1842 -n 1
top - 14:37:00 up 12 days,  3:50,  2 users,  load average: 0.41, 0.35, 0.39
Tasks:   1 total,   0 running,   1 sleeping,   0 stopped,   0 zombie

  PID USER      PR  NI    VIRT    RES    SHR S  %CPU  %MEM     TIME+ COMMAND
 1842 mouctar   20   0 2345672 185432  54212 S   3.8   1.1   2:17.55 node
```
