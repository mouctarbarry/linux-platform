## Syntaxe

```bash
history [OPTIONS] [N]
```

## Options principales

| Option | Description                                        |
| ------ | -------------------------------------------------- |
| `-c`   | Effacer tout l'historique en mémoire               |
| `-d N` | Supprimer l'entrée numero N de l'historique        |
| `-w`   | Ecrire l'historique courant dans le fichier        |
| `N`    | Afficher les N dernières commandes                 |

## Exemples

### Afficher les 10 dernières commandes

```bash
$ history 10
  491  cd /var/log
  492  tail -f syslog
  493  grep "error" syslog | wc -l
  494  systemctl status nginx
  495  sudo systemctl restart nginx
  496  curl -I http://localhost
  497  df -h
  498  free -m
  499  top
  500  history 10
```

### Supprimer une entrée spécifique de l'historique

```bash
$ history | grep "mot_de_passe"
  347  mysql -u root -pmot_de_passe_secret
$ history -d 347
$ history | grep "mot_de_passe"
$
```
