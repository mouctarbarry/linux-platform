## Syntaxe

```bash
lsblk [OPTIONS] [PERIPHERIQUE...]
```

## Options principales

| Option       | Description                                          |
| ------------ | ---------------------------------------------------- |
| `-f`         | Afficher les systèmes de fichiers et les UUID         |
| `-o COLONNES`| Sélectionner les colonnes à afficher                  |
| `-l`         | Afficher en format liste au lieu de l'arborescence    |
| `-d`         | Afficher uniquement les disques (sans les partitions) |

## Exemples

### Lister tous les périphériques de stockage

```bash
$ lsblk
NAME   MAJ:MIN RM   SIZE RO TYPE MOUNTPOINTS
sda      8:0    0 238.5G  0 disk
├─sda1   8:1    0   512M  0 part /boot/efi
├─sda2   8:2    0   230G  0 part /
└─sda3   8:3    0     8G  0 part [SWAP]
sdb      8:16   0   1.8T  0 disk
└─sdb1   8:17   0   1.8T  0 part /mnt/data
sr0     11:0    1  1024M  0 rom
```

### Afficher les systèmes de fichiers et UUID

```bash
$ lsblk -f
NAME   FSTYPE FSVER LABEL  UUID                                 MOUNTPOINTS
sda
├─sda1 vfat   FAT32 EFI    A1B2-C3D4                            /boot/efi
├─sda2 ext4   1.0          e5f6a7b8-c9d0-1234-5678-9abcdef01234 /
└─sda3 swap   1            12345678-abcd-ef01-2345-6789abcdef01 [SWAP]
sdb
└─sdb1 xfs                 abcdef01-2345-6789-abcd-ef0123456789 /mnt/data
```

### Afficher des colonnes personnalisées

```bash
$ lsblk -o NAME,SIZE,TYPE,FSTYPE,MOUNTPOINTS
NAME   SIZE TYPE FSTYPE MOUNTPOINTS
sda  238.5G disk
├─sda1 512M part vfat   /boot/efi
├─sda2 230G part ext4   /
└─sda3   8G part swap   [SWAP]
sdb    1.8T disk
└─sdb1 1.8T part xfs    /mnt/data
```
