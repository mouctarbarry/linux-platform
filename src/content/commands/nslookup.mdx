## Syntaxe

```bash
nslookup [OPTIONS] NOM [SERVEUR]
```

## Options principales

| Option          | Description                                  |
| --------------- | -------------------------------------------- |
| `-type=TYPE`    | Definir le type d'enregistrement DNS         |
| `-port=PORT`    | Specifier le port du serveur DNS             |
| `-timeout=SEC`  | Delai d'attente en secondes                  |
| `-debug`        | Activer le mode debogage                     |
| `-retry=N`      | Nombre de tentatives en cas d'echec          |

## Exemples

### Rechercher l'adresse IP d'un domaine

```bash
$ nslookup google.com
Server:         127.0.0.53
Address:        127.0.0.53#53

Non-authoritative answer:
Name:   google.com
Address: 142.250.75.238
```

### Interroger un serveur DNS spécifique

```bash
$ nslookup example.com 8.8.8.8
Server:         8.8.8.8
Address:        8.8.8.8#53

Non-authoritative answer:
Name:   example.com
Address: 93.184.216.34
```

### Rechercher les enregistrements MX d'un domaine

```bash
$ nslookup -type=mx gmail.com
Server:         127.0.0.53
Address:        127.0.0.53#53

Non-authoritative answer:
gmail.com       mail exchanger = 5 gmail-smtp-in.l.google.com.
gmail.com       mail exchanger = 10 alt1.gmail-smtp-in.l.google.com.
gmail.com       mail exchanger = 20 alt2.gmail-smtp-in.l.google.com.
```

### Effectuer une recherche DNS inverse

```bash
$ nslookup 8.8.8.8
Server:         127.0.0.53
Address:        127.0.0.53#53

Non-authoritative answer:
8.8.8.8.in-addr.arpa    name = dns.google.
```
